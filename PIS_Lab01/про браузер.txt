DOM - это модель представления браузером HTML страницы, API для доступа к древовидной структуры
браузер характеризуется: dom, bom (api), css, js
в bom входит local and session storage, объект history, window, navigator
xmlhttprequest - отдельный объект для асинхронных запросов из js
у html5 есть свой api (canvas, audio- video- filesystem-)
кукки - фрагмент данных ключ-значение, который может быть сохранен на стороне клиента по указанию сервера
